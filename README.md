##Automation Brain (MVP)

Automation Brain is a modular backend system that interprets natural language commands and executes automated tasks through a structured execution engine.

Built with a DevOps-oriented mindset, the system separates intent parsing, execution logic, and plugin-based actions to simulate a lightweight automation orchestrator.

## Key Features
Natural language command parsing (rule-based MVP layer)
Modular execution engine with action routing
Plugin-based architecture for extensible automation tasks
Centralized logging system for execution tracking
REST API built with Flask
Structured request → parse → execute workflow
##Architecture Overview

User Input → API Layer → Intent Parser → Action Router → Execution Engine → Plugin System → Logger
##Tech Stack
Python
Flask
Modular architecture design
dotenv for environment configuration
##Purpose

This project was built as a foundational step toward AI-powered automation systems and DevOps-style backend architectures. It focuses on clean separation of concerns, scalability, and execution reliability rather than just scripting automation.
